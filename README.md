# Django Calculator

A simple **web-based calculator** built using **Django**, **HTML**, **CSS**, and **JavaScript**.  
This calculator can perform addition, subtraction, multiplication, and division, and shows results instantly using JavaScript.

## Features
- Add, subtract, multiply, and divide two numbers
- Instant calculation without page reload (JavaScript)
- Styled with CSS for better user experience
- Beginner-friendly Django project
